Projects

RPG Combat
RPG Combat is a 3rd Person Role Playing Game that implements basic game user interfaces, combat mechanics, stats, and progression systems.

AI Ship Capstone
AI Ship Capstone is a VR project that allows the player to control a Colonial Ship using the steering wheel and uses Speech Recognition to control the speed of the ship. This project uses all different types of skills that I have learned throughout the Udacity Nanodegree Program.

Rube Goldberg Challenge
Rube Goldberg Game that challenges players to create contraptions to solve physics puzzles in Virtual Reality.

Puzzler
Puzzler is a Mobile Virtual Reality Game that shows the user a random pattern via lit up floating orbs that they must interact with in the correct sequence in order to proceed.
